# 第一幕：边境小镇的低语

浓雾如厚重的幕布，将迷雾村笼罩在一片朦胧之中。冒险者踩着泥泞的小路踏入村庄时，石板路两旁的木屋烟囱正冒着袅袅炊烟。空气中混杂着铁匠铺的煤烟味和醉龙酒馆飘来的麦酒香气。

## 场景一：村长的小屋

冒险者在村中央找到了老格雷姆的小屋。老人正坐在门前的摇椅上，手中摩挲着一枚锈迹斑斑的骑士勋章。他抬起头，浑浊的眼中闪过一丝审视。

*"你来了……我已经等了三十年了。不，也许更久。时间就像山间的小溪，你看着它流淌，却不知它何时改变了方向。"*

## 场景二：铁匠铺与酒馆

布隆德的铁锤在铁砧上敲出铿锵的节奏。他的铺子里摆满了各式武器和工具。

而在村子另一头，醉龙酒馆内，玫瑰玛丽正擦拭着永远擦不完的杯子，耳朵却竖听着每一桌的对话。

## 暗流

夜深时，冒险者或许会注意到村外闪烁的诡异灯火——那是暗影势力的探子在密林中活动。卡洛斯正为此忧心忡忡。

# 第二幕：幽暗密林的试炼

穿过迷雾村的北出口，一条被藤蔓遮蔽的小路蜿蜒伸入幽暗密林。这里的树木比村中任何建筑都要古老，枝叶交织成天然的穹顶，将阳光切割成碎片。

## 场景一：密林边缘

无名者的小屋隐藏在一片荆棘之后。这个蓬头垢面的老人不愿与任何人交谈，但如果冒险者展现出诚意——或者提到了"国王"二字——他可能会喃喃说出一些令人费解的话：

*"记忆……那些被偷走的记忆……王冠不是用来戴的，是用来关的……"*

## 场景二：女巫的谜题

密林深处，赛琳娜的小屋矗立在一棵巨大的古橡树下。屋内挂满了风干的草药和闪烁的水晶球。

她会提出三道谜题：

*"第一题：我没有翅膀，却能飞遍天涯；我没有声音，却能传遍四海。我是什么？"*（答案：传说/消息）

*"第二题：越是使用，越是富有。越是囤积，越是贫穷。我是什么？"*（答案：智慧/知识）

*"第三题：我诞生于光明，却只能存在于黑暗。阳光是我的父亲，但一见我便消散。我是什么？"*（答案：影子）

三道谜题全部答对后，赛琳娜会指向密林最深处——月光下，遗迹的入口会显现出来。

## 场景三：偶遇

在密林中，冒险者可能会遇到狐狸杰克——他正躲在一棵大树后，鬼鬼祟祟地观察着什么。他声称发现了暗影势力的营地，想和冒险者做笔交易。

# 第三幕：遗迹深处的真相

遗迹的大门由黑色大理石构成，上面刻满了古代魔法符文。当月光照射其上时，符文会发出幽蓝色的光芒，石门缓缓开启。

## 场景一：前厅

遗迹内部是一个巨大的地下殿堂。墙壁上镶嵌着发出微光的宝石，照亮了四周。前厅中央有一座石台，上面放着一卷泛黄的羊皮纸——那是最后一位国王的遗书。

遗书内容：

*"致后来者：*
*当你读到这封信时，说明封印已经开始松动。我并非失踪，而是自愿走入这座遗迹。*
*王冠中封印的，不是邪恶，而是整个王国的记忆。三百年前，一场名为'遗忘之灾'的浩劫即将降临——它会将所有人的记忆吞噬殆尽，让文明倒退千年。*
*唯一的方法，是将所有记忆封印于王冠之中。我选择牺牲自己，成为这封印的守护者。*
*如今，暗影势力企图打破封印，释放这些记忆——但他们不知道，记忆的洪流同样会摧毁这个已经安定了三百年的世界。*
*选择权在你：保持封印，或者释放真相。"*

## 场景二：守护者的考验

国王的灵魂以幻影的形式出现在冒险者面前。他不会战斗，而是会问一个问题：

*"告诉我，冒险者——你认为一个没有过去的文明，和一个被过去的重量压垮的文明，哪一个更可悲？"*

冒险者的回答将决定最终的结局。

## 结局分支

**结局一：保持封印**
冒险者选择维持封印，离开遗迹。王国继续和平，但真相永远被埋葬。

**结局二：释放记忆**
冒险者选择释放王冠中的记忆。三百年的历史涌入每个人的脑海——人们看到了王国的辉煌，也看到了那场几乎毁灭一切的灾难。世界将永远改变。

**结局三：折中之路**
如果冒险者足够智慧，或许能找到第三种方式——将记忆缓慢释放，让世界有时间适应。
